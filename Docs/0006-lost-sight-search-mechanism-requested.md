# 0006. 시야 이탈 시 "재탐색(search)" 메커니즘 추가 요청

## 날짜
2026-07-07

## 요청
지금 적유닛 공격선택일때 시야에서 사라지고 그 위치로 Move 하는게 좋을거 같아 중간에 다른 적유닛을 만나면 그녀석을 공격하네 그리고 시야에서 사라져서 마지막 위치를 move로 가고 거기서 다시 그 적유닛을 search하는 느낌으로 하고 만약 search했는데 시야범위에 없다. 그러면 그때 다른 공격 유닛을 공격하는식으로 구현해줘

## 답변 / 변경사항 (이후 0007에서 되돌림)
3단계 추격/재탐색 메커니즘으로 구현:
1. **추격(`orderedTarget`)**: 대상이 살아있고 사거리 안이면 계속 쫓아가며 다른 적은 무시.
2. **시야 이탈 → 그냥 이동**: 대상을 놓치면 `lostTarget`에 저장하고 마지막 위치로 `Move` 상태로 이동(다른 적과 교전하지 않음).
3. **도착 후 재탐색**: 도착 지점에서 원래 대상이 `searchRadius`(기본 10) 안에 있는지 확인 → 재발견하면 추격 재개, 못 찾으면 완전히 포기하고 `Idle`(평범한 자동 교전 모드)로 전환.
